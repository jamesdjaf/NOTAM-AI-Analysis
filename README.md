# AI Flight Operations Decision Support Platform

## Vision

An AI-powered operational decision-support platform that assists airline Flight Dispatchers in analysing NOTAMs against Operational Flight Plans (OFPs).

---

## Current MVP

- OFP Parser
- ICAO NOTAM Parser
- Route Conflict Detection
- Operational Recommendations
